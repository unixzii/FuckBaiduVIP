# Introduction

支持对歌曲标题、表演者、专辑的模糊搜索。对搜索内容可以直接进行下载，个别需要 VIP 身份的歌曲也可以直接下载。使用时请注意版权。

本程序为面向 OS X 的 Xcode 项目，下载工程后可以直接构建，无依赖。

所使用的 API 可能随时变动，作者并不能保证该项目的时效性，理论上仅供研究。
